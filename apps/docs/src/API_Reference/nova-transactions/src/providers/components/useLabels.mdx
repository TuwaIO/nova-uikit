# useLabels()

Defined in: [packages/nova-transactions/src/providers/LabelsProvider.tsx:42](https://github.com/TuwaIO/nova-uikit/blob/9a26710ac8b30dc3dfd19fe675eade473607bcdf/packages/nova-transactions/src/providers/LabelsProvider.tsx#L42)

A custom hook to easily access the i18n labels from any component
within the `LabelsProvider` tree.

## Returns

[`TuwaLabels`](../../type-aliases/TuwaLabels.md)

The complete object of UI labels.

## Example

```ts
const MyComponent = () => {
const labels = useLabels();
return <h1>{labels.walletModal.title}</h1>;
}
```
