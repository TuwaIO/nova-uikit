# useHasNovaConnectContext()

> **useHasNovaConnectContext**(): `boolean`

Defined in: [packages/nova-connect/src/hooks/useNovaConnect.ts:130](https://github.com/TuwaIO/nova-uikit/blob/465727cc7679afe03481c0a87226bb585ac2e3b7/packages/nova-connect/src/hooks/useNovaConnect.ts#L130)

Hook to check if NovaConnect context is available

## Returns

`boolean`

True if context is available, false otherwise

## Example

```typescript
function ConditionalComponent() {
  const hasContext = useHasNovaConnectContext();

  if (!hasContext) {
    return <div>NovaConnect provider not found</div>;
  }

  return <ConnectedComponent />;
}
```
