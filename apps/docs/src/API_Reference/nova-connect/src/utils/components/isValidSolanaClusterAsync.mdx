# isValidSolanaClusterAsync()

> **isValidSolanaClusterAsync**(`cluster`): `Promise`\<`boolean`\>

Defined in: [packages/nova-connect/src/utils/getChainsListByConnectorType.ts:375](https://github.com/TuwaIO/nova-uikit/blob/fa4eae2e64decd5a28150a0f1976e8dd4af9d026/packages/nova-connect/src/utils/getChainsListByConnectorType.ts#L375)

Validates whether a cluster name is valid for Solana connections.

This function loads the Solana adapter and uses it to validate cluster
names against known Solana cluster monikers and custom configurations.

## Parameters

### cluster

`string`

Cluster name to validate

## Returns

`Promise`\<`boolean`\>

Promise resolving to true if cluster is valid

## Example

```typescript
const isValid = await isValidSolanaCluster('mainnet-beta'); // true
const isInvalid = await isValidSolanaCluster('invalid-cluster'); // false

// Use for form validation
if (await isValidSolanaCluster(userInput)) {
  // Proceed with connection
} else {
  // Show error message
}
```
