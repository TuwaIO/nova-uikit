# getChainName()

> **getChainName**(`chainId`): [`ChainInfo`](../interfaces/ChainInfo.md)

Defined in: [packages/nova-core/src/utils/getChainName.ts:62](https://github.com/TuwaIO/nova-uikit/blob/c6264598fe8e3f231a0d8884c06f8c0b311f3334/packages/nova-core/src/utils/getChainName.ts#L62)

Retrieves blockchain network information based on its chain ID.

Supports both EVM and non-EVM chain identifiers:
- **Numeric IDs (EVM):** e.g., `1` → "Ethereum", `137` → "Polygon"
- **String IDs (Non-EVM):** e.g., `"solana:devnet"` → "Solana Devnet"

For string IDs with environment suffixes (like "solana:devnet"),
the suffix is appended to the name if it's a known dev/test environment.

## Parameters

### chainId

Chain identifier (number for EVM, string for non-EVM)

`string` | `number`

## Returns

[`ChainInfo`](../interfaces/ChainInfo.md)

Chain information object with name, id, and original chainId

## Example

```ts
// EVM network
getChainName(1)
// → { name: "Ethereum", id: "ethereum", chainId: 1 }

// Solana devnet
getChainName("solana:devnet")
// → { name: "Solana Devnet", id: "solana", chainId: "solana:devnet" }

// Unknown network
getChainName(999999)
// → { name: "Unknown", id: "unknown", chainId: 999999 }
```
