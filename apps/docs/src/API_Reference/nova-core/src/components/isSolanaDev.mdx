# isSolanaDev()

> **isSolanaDev**(`chainId`): `boolean`

Defined in: [packages/nova-core/src/utils/isSolanaDev.ts:10](https://github.com/TuwaIO/nova-uikit/blob/c69ad9ed784ed588d7138f53d4e3088d4951075f/packages/nova-core/src/utils/isSolanaDev.ts#L10)

Determines if the current chain is a Solana development or test network.

It strictly checks if the ID is a string containing 'solana' AND
indicates a non-production environment ('devnet' or 'testnet').

## Parameters

### chainId

The chain identifier (e.g., "solana:devnet", 1, "ethereum").

`string` | `number`

## Returns

`boolean`

True only if it is a Solana dev/test chain.
