## What is Nova?

Nova is a collection of React component libraries built on a shared core. It provides everything from foundational utilities and hooks in `@tuwaio/nova-core` to specialized, feature-rich components in packages like `@tuwaio/nova-transactions`. The entire system is designed to be cohesive, themeable, and easy to integrate into any React project.

### Core Philosophy

-   **Modularity**: Start with `@tuwaio/nova-core` for essential utilities and theming, then add other `nova-*` packages as you need them.
-   **Developer Experience**: Simple, predictable APIs with full TypeScript support make development a breeze.
-   **Customization**: A powerful theming system using CSS variables allows you to tailor the look and feel to match your brand perfectly.
-   **Accessibility First**: Built on top of Radix UI primitives to ensure components are accessible out-of-the-box.

## üöÄ Getting Started

### 1. Installation

Install the Nova packages you need. For transaction tracking, you'll typically want both.

```bash
# Using pnpm
pnpm add react-toastify framer-motion @radix-ui/react-dialog @heroicons/react @bgd-labs/react-web3-icons @tuwaio/pulsar-core @tuwaio/nova-core dayjs ethereum-blockies-base64 react immer zustand clsx tailwind-merge

# Using npm
npm install react-toastify framer-motion @radix-ui/react-dialog @heroicons/react @bgd-labs/react-web3-icons @tuwaio/pulsar-core @tuwaio/nova-core dayjs ethereum-blockies-base64 react immer zustand clsx tailwind-merge

# Using yarn
yarn add react-toastify framer-motion @radix-ui/react-dialog @heroicons/react @bgd-labs/react-web3-icons @tuwaio/pulsar-core @tuwaio/nova-core dayjs ethereum-blockies-base64 react immer zustand clsx tailwind-merge
````

### 2. Setup CSS

Create a main CSS file (e.g., `src/styles/app.css`) and import Nova's styles **before** Tailwind's utility classes.

```css
/* src/styles/app.css */
/* Import Nova's core theme variables and styles */
@import "@tuwaio/nova-core/dist/index.css";
@import "@tuwaio/nova-transactions/dist/index.css"; /* If using this package */
/* Import Tailwind's base, components, and utilities */
@import "tailwindcss";
```

Then, import this CSS file into your main application entry point (e.g., `layout.tsx` or `main.tsx`).

## üé® Theme Customization

Nova is built on a robust system of CSS variables. To apply your custom theme, simply override them in your global stylesheet. **Make sure your overrides come after the `@import` of Nova's styles.**

```css
/* src/styles/app.css */
@import "@tuwaio/nova-core/dist/index.css";
@import "@tuwaio/nova-transactions/dist/index.css";
@import "tailwindcss";

/* 3. Your custom theme overrides */
:root {
  --tuwa-text-accent: theme('colors.blue.500');
  --tuwa-text-primary: theme('colors.slate.800');
  --tuwa-bg-primary: theme('colors.white');
  --tuwa-border-primary: theme('colors.slate.200');
  /* ... add other overrides as needed */
}

.dark {
  --tuwa-text-accent: theme('colors.blue.400');
  --tuwa-text-primary: theme('colors.slate.100');
  --tuwa-bg-primary: theme('colors.slate.900');
  --tuwa-border-primary: theme('colors.slate.700');
  /* ... etc. */
}
```
