## What is Nova?

Nova is a collection of React component libraries built on a shared core. It provides everything from foundational utilities and hooks in `@tuwaio/nova-core` to specialized, feature-rich components in packages like `@tuwaio/nova-transactions`. The entire system is designed to be cohesive, themeable, and easy to integrate into any React project.

## Core Philosophy

-   **Modularity**: Start with `@tuwaio/nova-core` for essential utilities and theming, then add other `nova-*` packages as you need them.
-   **Developer Experience**: Simple, predictable APIs with full TypeScript support make development a breeze.
-   **Customization**: A powerful theming system using CSS variables allows you to tailor the look and feel to match your brand perfectly.
-   **Headless-First**: While we provide beautiful pre-styled components, the core logic is often available through headless hooks for maximum flexibility.

## Getting Started

### 1. Installation

Every Nova project starts with `@tuwaio/nova-core`. You can then add other packages as needed.

```bash
pnpm add @tuwaio/nova-core @tuwaio/nova-transactions
````

You will also need to have peer dependencies like `react` and `tailwindcss` installed.

### 2\. Setup

With Tailwind CSS, setup is minimal. Create a main CSS file (e.g., `src/styles/app.css`) and import Tailwind's base styles along with Nova's core theme variables.

```css
/* src/styles/app.css */

/* Import Tailwind's base, components, and utilities */
@import "tailwindcss";

/* Import Nova's core theme variables and styles */
@import "@tuwaio/nova-core/dist/index.css";
@import "@tuwaio/nova-transactions/dist/index.css"; /* If using this package */
```

Then, import this CSS file into your main application entry point (e.g., `main.tsx` or `layout.tsx`).

### 3\. Basic Usage Example

Here‚Äôs a simple example of using a hook from `@tuwaio/nova-core` and styling a button with Nova's theme variables.

```tsx
// src/components/MyComponent.tsx
import { useState } from 'react';
import { useCopyToClipboard } from '@tuwaio/nova-core';

function MyComponent() {
  const [copied, copy] = useCopyToClipboard();

  return (
    <div className="p-4">
      <h1 className="text-lg font-bold text-[var(--tuwa-text-primary)]">
        Welcome to Nova UI Kit
      </h1>
      <button
        onClick={() => copy('Hello, Nova!')}
        className="mt-2 px-4 py-2 bg-[var(--tuwa-bg-muted)] text-[var(--tuwa-text-secondary)] rounded-md hover:bg-[var(--tuwa-border-primary)]"
      >
        {copied ? 'Copied to Clipboard!' : 'Copy Text'}
      </button>
    </div>
  );
}
```

## Theme Customization

Nova is built on a robust system of CSS variables. Simply override them in your global stylesheet to apply your custom theme. Make sure your overrides come after the `@import` of Nova's styles.

```css
/* src/styles/app.css */
@import "tailwindcss";
@import "@tuwaio/nova-core/dist/index.css";
@import "@tuwaio/nova-transactions/dist/index.css";

/* Your custom theme overrides */
:root {
  --tuwa-text-accent: theme('colors.teal.500');
  --tuwa-text-primary: theme('colors.slate.800');
  --tuwa-bg-primary: theme('colors.white');
  --tuwa-border-primary: theme('colors.slate.200');
  /* ... add other overrides as needed */
}

.dark {
  --tuwa-text-accent: theme('colors.teal.400');
  --tuwa-text-primary: theme('colors.slate.100');
  --tuwa-bg-primary: theme('colors.slate.900');
  --tuwa-border-primary: theme('colors.slate.700');
  /* ... etc. */
}
```
